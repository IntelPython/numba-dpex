Info commands¶

info functions¶

Displays the list of functions in the debugged program.

Syntax¶

info functions
info functions [Regex]

Parameters¶

Regex

If specified, the info functions command lists the functions matching the regex. If omitted, the command lists all functions in all loaded modules (main program and shared libraries).

Note

Running the info functions command without arguments may produce a lot of output as the list of all functions in all loaded shared libraries is typically very long.

Consider Numba-dppy kernel code. See the source file numba_dppy/examples/debug/simple_sum.py:

15import numpy as np
16import numba_dppy as dppy
17import dpctl
18
19
20@dppy.kernel(debug=True)
21def data_parallel_sum(a, b, c):
22    i = dppy.get_global_id(0)
23    c[i] = a[i] + b[i]
24
25
26global_size = 10
27N = global_size
28
29a = np.array(np.random.random(N), dtype=np.float32)
30b = np.array(np.random.random(N), dtype=np.float32)
31c = np.ones_like(a)
32
33with dpctl.device_context("opencl:gpu"):
34    data_parallel_sum[global_size, dppy.DEFAULT_LOCAL_SIZE](a, b, c)
35
36print("Done...")
