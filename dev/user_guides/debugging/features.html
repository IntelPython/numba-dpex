Supported Features

Numba-dpex and Intel® Distribution for GDB* provide at least following debugging features:

    • break function

    • Breakpoints with nested functions

    • step
    • stepi
    • next
    • set scheduler-locking step
  • Information About a Frame
    • info args
    • info locals
  • Backtrace
