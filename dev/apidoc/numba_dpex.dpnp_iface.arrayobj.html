numba_dpex.dpnp_iface.arrayobj module

numba_dpex.dpnp_iface.arrayobj.ol_dpnp_empty(shape, dtype=None, usm_type=None, device=None, sycl_queue=None)

Implementation of an overload to support dpnp.empty inside a jit function.

Args:

shape (tuple): Dimensions of the array to be created. dtype optional): Data type of the array. Can be typestring,

a numpy.dtype object, numpy char string, or a numpy scalar type. Default: None

usm_type (“device”|”shared”|”host”, optional): The type of SYCL USM

allocation for the output array. Default: “device”.

device (optional): array API concept of device where the output array

is created. device can be None, a oneAPI filter selector string, an instance of dpctl.SyclDevice corresponding to a non-partitioned SYCL device, an instance of dpctl.SyclQueue, or a Device object returnedby dpctl.tensor.usm_array.device. Default: None.

sycl_queue (dpctl.SyclQueue, optional): Not supported.

Returns: Numba implementation of the dpnp.empty
