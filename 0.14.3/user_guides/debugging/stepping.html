Stepping¶

Note

Known issues:

  • Stepping works correctly only for kernels. Nested functions can cause problems.

Consider numba-dppy kernel code:

 1import numpy as np
 2import numba_dppy as dppy
 3import dpctl
 4
 5@dppy.kernel
 6def data_parallel_sum (a, b, c):
 7    i = dppy.get_global_id (0)  # numba-kernel-breakpoint
 8    l1 = a[i]                   # second-line
 9    l2 = b[i]                   # third-line
10    c[i] = l1 + l2              # fourth-line
11
12global_size = 10
13N = global_size
14a = np.array(np.random.random(N), dtype=np.float32)
15b = np.array(np.random.random(N), dtype=np.float32)
16c = np.ones_like(a)
17
18with dpctl.device_context("opencl:gpu") as gpu_queue:
19    data_parallel_sum[global_size, dppy.DEFAULT_LOCAL_SIZE](a, b, c)
