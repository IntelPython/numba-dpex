Random Number Generation#

Numba-dpex does not provide a random number generation algorithm that can be executed on the SYCL-supported Device.

Numba-dpex provides access to NumPy random algorithms that can be executed on the SYCL-supported device via integration with dpnp Random.

Supported functions#

Simple random#

  • random

  • sample

  • ranf

  • random_sample

  • rand

  • randint

  • random_integers

Distribution#

  • beta

  • binomial

  • chisquare

  • exponential

  • gamma

  • geometric

  • gumbel

  • hypergeometric

  • laplace

  • lognormal

  • multinomial

  • multivariate_normal

  • negative_binomial

  • normal

  • poisson

  • rayleigh

  • standard_cauchy

  • standard_exponential

  • standard_gamma

  • standard_normal

  • uniform

  • weibull
