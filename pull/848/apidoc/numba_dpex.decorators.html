The decorator to write a numba_dpex kernel function.

A kernel function is conceptually equivalent to a SYCL kernel function, and gets compiled into either an OpenCL or a LevelZero SPIR-V binary kernel. A dpex kernel imposes the following restrictions:

  • A numba_dpex.kernel function can not return any value.

  • All array arguments passed to a kernel should be of the same type and have the same dtype.
