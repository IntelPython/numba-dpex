numba_dpex.core.typeconv.to_usm_ndarray(suai_attrs, addrspace=1)

Converts an array-like object that has the _sycl_usm_array_interface__ attribute to numba_dpex.types.UsmNdArray.

Args:

suai_attrs: The extracted SUAI information for an array-like object. addrspace: Address space this array is allocated in.

Returns: The Numba type for SUAI array.

Raises:

NotImplementedError: If the dtype of the passed array is not supported.
